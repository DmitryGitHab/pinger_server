Установка зависимостей:
"pip install -r req.txt"

запуск сервера:
'uvicorn main:app --reload'

Принцип работы, при старте сервера, осуществляется обращение к Базе данных, если она пустая на странице сайта будет пусто, если в БД есть сервера они отобразятся на главной стриничке после пингования и получения актуальных данных по ним.
Сервера можно добавлять вручную, можно экспортировать из csv файла (сервера сохраняться в БД)
Соответсвенно при удалении со страницы серверов они так же удаляются из БД.
Так же  актуальную информацию по серверам можно экспортировать в CSV Файл.
